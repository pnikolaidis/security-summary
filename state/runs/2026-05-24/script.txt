Andi here with your Sunday catch-up. Three security stories worth your attention today — one active exploitation campaign, one supply chain hit on a popular developer ecosystem, and a quick patch for a tool most of us use daily.

First up: Ghost CMS is being actively exploited right now. Attackers have weaponized a critical SQL injection flaw, tracked as CVE-2026-26980, and are running a large-scale campaign that injects malicious JavaScript into vulnerable sites. Once that code lands on a visitor's browser, it triggers what's known as a ClickFix attack — a social engineering trick that prompts the user to run attacker-controlled commands on their own machine. This is a full chain from server compromise to end-user infection, and the campaign appears to be automated and widespread. If you run Ghost CMS anywhere, patch it today and audit your theme files and stored content for any injected scripts.

Second story: a supply chain attack hit the Laravel ecosystem. Attackers hijacked the Laravel Lang localization packages — widely used PHP libraries pulled through Composer by millions of developers — and used a clever trick: instead of modifying the source code directly, they abused GitHub version tags to push a malicious release. That approach slips past a lot of standard code review. The payload is a credential-stealing malware that targets developer machines and could exfiltrate application secrets, database credentials, or API keys. Any Laravel developer who updated Lang packages recently should treat their environment as potentially compromised, audit logs for unusual outbound connections, and rotate credentials as a precaution.

Third: Wireshark 4.6.6 is out today. This is a routine maintenance release, but it patches one security vulnerability alongside eleven bug fixes. The SANS Internet Storm Center flagged it as worth attention, particularly for analysts who open packet captures from untrusted sources — a crafted PCAP file could trigger the vulnerability. If Wireshark is part of your daily toolkit, take five minutes to update.

That's your digest. Stay safe out there.
